# Notebooks
Jupyter notebooks for Data Science related projects.

## Usage
Python, pip, and Jupyter Lab are required to run each notebook. Other libraries and tools vary per notebook.

Commonly used libraries:
1. numpy
2. pandas
3. matplotlib
4. scikit-learn
5. pytorch

It is recommended to create a Python virtual environment in the root directory of the repository for installing packages and running Jupyter Lab in a contained environment.
